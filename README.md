# CIgFlow_Livedemo
#

A Flow Matching-based generative framework for designing
antigen-conditioned antibody sequences using protein language
model embeddings.

## Features

- Antigen-conditioned generation
- Flow Matching latent diffusion
- Adjustable temperature
- FASTA export
- Streamlit web interface

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
